## Getting started

Start by installing all dependencies with `yarn install` or the `npm` equivalent.

To use the development server with hot reloading:

```bash
yarn dev
# or
npm run dev
```

To start the server in a production environment:

```bash
yarn start
# or
npm run start
```

To run the included tests:

```bash
yarn test
# or
npm run test
```
